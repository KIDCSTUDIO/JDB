1
注册 Cloudflare 账号

打开 cloudflare.com，用邮箱注册（免费，不需要绑卡）。

2
安装 Node.js

打开 nodejs.org，下载左边的 LTS 版本，一路点「下一步」安装。装完后验证：Win+R 输入 cmd 回车，输入 node -v 看到版本号就是成功了。

3
创建 Worker 项目

在你想放代码的地方（比如桌面），打开 cmd，逐行执行：

mkdir checklist-worker
cd checklist-worker
npm create cloudflare@latest -- --name checklist-parser -y
cd checklist-parser

选 Hello World Worker / JavaScript 即可。提示登录时按回车，会自动打开浏览器授权。

4
替换 Worker 代码

用下面这段完整代码替换 src/index.js 的全部内容（文件在刚创建的 checklist-parser/src/ 目录下，右键用记事本打开）：

/**
 * 产前样复核检点表 — AI 解析 Worker
 * 代理 DeepSeek API，供前端 HTML 调用
 */
export default {
  async fetch(request, env) {
    if (request.method === 'OPTIONS') {
      return new Response(null, { headers: corsHeaders() });
    }

    const url = new URL(request.url);
    if (url.pathname !== '/parse' || request.method !== 'POST') {
      return new Response('Not Found', { status: 404, headers: corsHeaders() });
    }

    try {
      const body = await request.json();
      const { items: rawItems, project } = body;
      if (!rawItems || !project) {
        return Response.json({ error: '缺少参数' }, { status: 400, headers: corsHeaders() });
      }

      const apiKey = env.DEEPSEEK_API_KEY || request.headers.get('x-api-key') || '';
      if (!apiKey) {
        return Response.json({ error: '请在 HTML 中填写 API Key' }, { status: 401, headers: corsHeaders() });
      }

      const dsResponse = await fetch('https://api.deepseek.com/v1/chat/completions', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${apiKey}` },
        body: JSON.stringify({
          model: 'deepseek-reasoner',
          messages: [{ role: 'user', content: SYSTEM_PROMPT + '\n\n---\n\n请解析以下用户输入：\n项目名：' + project + '\n' + rawItems }],
          max_tokens: 4096,
        }),
      });

      if (!dsResponse.ok) {
        const errText = await dsResponse.text();
        return Response.json({ error: 'DeepSeek API 错误 (' + dsResponse.status + '): ' + errText.slice(0, 200) }, { status: 502, headers: corsHeaders() });
      }

      const dsData = await dsResponse.json();
      const content = dsData.choices?.[0]?.message?.content || '';
      let parsed;
      try {
        parsed = JSON.parse(content.replace(/^```json\s*/, '').replace(/```\s*$/, '').trim());
      } catch (e) {
        return Response.json({ error: '解析 AI 返回失败', raw: content.slice(0, 500) }, { status: 500, headers: corsHeaders() });
      }
      if (!Array.isArray(parsed.items)) {
        return Response.json({ error: 'AI 返回格式异常' }, { status: 500, headers: corsHeaders() });
      }
      return Response.json(parsed, { headers: corsHeaders() });
    } catch (err) {
      return Response.json({ error: 'Worker 内部错误: ' + err.message }, { status: 500, headers: corsHeaders() });
    }
  },
};

function corsHeaders() {
  return {
    'Access-Control-Allow-Origin': '*',
    'Access-Control-Allow-Methods': 'POST, OPTIONS',
    'Access-Control-Allow-Headers': 'Content-Type, x-api-key',
    'Content-Type': 'application/json',
  };
}

const SYSTEM_PROMPT = `你是家具制造行业的品质工程师。用户会给你项目名称和口语化的检点项列表，请你将其解析为结构化的产前样复核检点表。

## 输出格式（严格 JSON）
{
  "items": [
    {
      "part": "部位/配件（核心部件名）",
      "check_item": "重点复核项目（描述要复核的核心要点）",
      "standard": "判定标准（具体尺寸数据或合格状态描述）",
      "method": "复核方法（卡尺/卷尺测量、目视检查、手检牢固度等）"
    }
  ]
}

## 解析规则
### 部位/配件：提取文本中的核心家具部件名称
### 重点复核项目：将口语化描述转为正式表述
### 判定标准：有尺寸保留尺寸，外观类描述表面标准，无明确标准写"按确认版图纸及标准执行"
### 复核方法：含尺寸→卡尺/卷尺测量；表面→目视检查手触；安装→模拟安装操作`;
5
部署上线

在 cmd 中确保当前在项目目录下（checklist-parser），执行：

npx wrangler deploy

部署成功后，终端会显示一行绿色的 URL，类似 https://checklist-parser.xxxx.workers.dev，把它复制下来。

6
填回工具

把刚才复制的 URL 粘贴到上方「Worker URL」输入框（记得末尾带 /parse）。再去 platform.deepseek.com 创建一个 API Key，粘贴到「API Key」输入框。搞定 🎉

检点项输入 — 粘贴口语化内容，选择解析方式
解析结果 — 点击单元格直接编辑，G-K列留空待填
序号 适用款号 部位/配件 重点复核项目 判定标准 复核方法 复核结果 实测值/问题说明 整改责任人 完成日期 复核签名
请输入项目名和检点项，然后点击「解析」
