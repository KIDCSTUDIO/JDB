# 产前样复核检点表生成器

家具制造品质检点表生成工具 · 手机/电脑都能用 · 离线可用 · AI 智能解析

## 📱 使用方式

1. 用浏览器打开 GitHub Pages 链接
2. 点击地址栏「安装」→ 添加到桌面
3. 打开即全屏 App，断网也能用

## 🛠 功能

- **AI 解析**：输入口语化描述 → DeepSeek 自动生成检点表
- **规则解析**：离线模式，纯前端解析引擎
- **预览 & 导出**：一键生成 Excel（含 Sheet 2 追溯表）
- **草稿自动保存**：浏览器 localStorage，刷新不丢

## 🔧 本地开发

直接在浏览器打开 `index.html` 即可。
